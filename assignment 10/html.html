PIAIC

Presidential Initiative for Artificial Intelligence and Computing

About PIAIC

PIAIC is a privately funded, not-for-profit educational initiative started by the President of Pakistan, Dr. Arif Alvi, Kazi Rahat Ali, and Zia Khan to help promote education, research, and business opportunities in emerging tech fields like:

  • Cloud Native Computing
  • Artificial Intelligence
  • Blockchain
  • Internet of Things

Programs Offered

Cloud Native Computing

The Cloud Native Computing program prepares students to program in open source technologies like:

  • Linux
  • Docker
  • Kubernetes
  • JavaScript
  • Node.js
  • React and React Native

The program also introduces the development of Microservices and software development techniques on cloud platforms like AWS.

Artificial Intelligence

The Artificial Intelligence program of PIAIC covers the foundations of AI technology, teaching programming in open-source languages such as:

  • Python
  • Numpy
  • Pandas
  • TensorFlow 2.0
  • Flask

Students will learn to deploy practical AI solutions and use cases, ending with real-life examples of modern-day AI in practice.

Blockchain

The Blockchain program emphasizes understanding blockchains like Ethereum and Hyperledger (Fabric) and learning to program in languages such as:

  • Solidity
  • JavaScript

Students will also develop Distributed Apps (DApps), explore frameworks like Web3 and Truffle, and gain an in-depth understanding of Blockchain-as-a-Service (BaaS), along with developing and offering an Initial Coin Offering (ICO) of a crypto-token.

Internet of Things

The Internet of Things program introduces students to open-source development technologies like:

  • Rust
  • Embedded Rust
  • WebAssembly
  • Hyper
  • Diesel
  • Google Assistant
  • Kubernetes

Students will become proficient in developing Microservices for IoT Edge, IoT Cloud, and voice computing for device control.
